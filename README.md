### Day1 사칙연산
- 두 수의 합
- 두 수의 차
- 두 수의 곱
- 몫 구하기

### Day2 사칙연산, 조건문
- 숫자 비교하기
- 나머지 구하기
- 나이 출력
- 두 수의 나눗셈

- 각도기
- 짝수의 합
- 배열의 평균값
- 양꼬치

- 삼각형의 완성조건(1)
- 배열 뒤집기
- 배열 두 배 만들기

### Day5
- 최댓값 만들기(1)
- 머쓱이보다 키 큰 사람
- 배열 자르기

### Day6
- 피자 나눠 먹기 (3)
- 중복된 숫자 개수
- 배열 원소의 길이
